# **Overview**

## **Features**
- 한강 공원을 중심으로 함께 달릴 크루원을 모집할 수 있습니다.

- 원하는 조건으로 크루를 검색하고, 참여할 수 있습니다.
- 실시간 크루원들과 채팅을 할 수 있습니다.
- 여러 조건에 따라 메달을 획득할 수 있습니다.
